Question008 - DAU 和 QPS 是什麼 ?
=====
* ### 每日活躍用戶 (DAU, Daily active users)。
* ### 每秒查詢次數 (QPS, Query per second)。
* ### 假設
    * ### 每個月 3 億活躍使用者。
    * ### 每天 50 % 使用。
    * ### 每人一天平均 2 則貼文。
    * ### 10 % 的堆文中包含多媒體。
    * ### 資料儲存五年。
* ### 估算
    * ### 每日活躍用戶 (DAU, Daily active users) = 3 * 0.5 = 1.5 億。
    * ### 推文每秒查詢次數 (QPS, Query per second) = 1.5 億 * 2 / 24 / 3600 s = 3500。
    * ### QPS 峰值 = 2 * 3500 = 7000。
