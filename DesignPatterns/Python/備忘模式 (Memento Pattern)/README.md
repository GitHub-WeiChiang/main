備忘模式 (Memento Pattern)
=====
* ### Capture the object's internal state without exposing its internal structure, so that the object can be returned to this state later.
* ### 在不破壞內部結構的前提下捕獲一個物件的內部狀態，這樣便可在以後將該物件恢復到原先保存的狀態。
